# Smart Match Booking - Event Booking Website

## Features
- User registration, login, and dashboard
- Event listing and details
- Booking with (mock) payment and promotion code
- E-ticket with QR code
- Admin panel for event management
- Analytics and promotion tracking
- Modern responsive UI (Bootstrap)

## Setup
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the app:
   ```bash
   python app.py
   ```
3. Access the site at [http://127.0.0.1:5000](http://127.0.0.1:5000)

- Default admin: `admin` / `admin`

## Project Structure
- `app.py` - Main Flask app
- `templates/` - HTML templates
- `static/css/` - Custom styles
- `static/js/` - JavaScript
- `instance/app.db` - SQLite database
